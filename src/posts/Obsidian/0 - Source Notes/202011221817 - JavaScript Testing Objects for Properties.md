# JavaScript Testing Objects for Existence of Properties

The ff can be used to check if the property of a given object exists or not:
- `.hasOwnProperty(propname)` method
- `in` keyword
```js
users.hasOwnProperty('Alan');
'Alan' in users;
// both return true
```
