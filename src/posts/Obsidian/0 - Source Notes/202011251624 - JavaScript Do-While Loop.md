# JavaScript Do-While Loop

```js
var ourArray = [];
var i = 0;
do {
  ourArray.push(i);
  i++;
} while (i < 5);
```

- ensures that the code inside loop is executed at least once
	- will continue to execute the code inside `while` the specified condition evalutates to true
